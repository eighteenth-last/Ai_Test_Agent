🔍 常见失败原因：

  1. 元素定位失败
    - 如某步点击按钮时索引不存在或页面未渲染完成。
    - 修复建议：增加等待机制（如 wait_for_element）、使用更稳定的定位方式（如 CSS Selector 或 XPath）。

    - 修复建议：设置页面加载超时阈值，加入 URL 变化监听器。

    - 修复建议：统一文案规范，或在断言中支持模糊匹配（正则表达式）。

    - 修复建议：添加窗口切换或关闭弹窗逻辑。
